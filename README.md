# Professional Notes App (Flutter)

A sleek, efficient, and professional-functional note-taking application built with Flutter. Designed for high information density and snappy performance.

## Features
- **Full CRUD:** Create, Read, Update, and Delete notes seamlessly.
- **Local Persistence:** All notes are stored securely on your device using SQLite.
- **Modern UI:** Clean, "Linear-style" interface with comfortable spacing and fluid transitions.
- **Dark Mode:** Native support for dark and light themes.
- **Responsive:** Optimized for a smooth experience across different screen sizes.

## Tech Stack
- **Framework:** Flutter (Stable)
- **State Management:** Provider
- **Local Storage:** SQLite (`sqflite`)
- **Testing:** Unit & Widget Tests

## Installation
1. Clone the repository.
2. Ensure Flutter is installed: `flutter doctor`.
3. Install dependencies: `flutter pub get`.
4. Run the app: `flutter run`.

## Design Philosophy
This app follows the **Professional-Functional** design language:
- **Clarity over Clutter:** No unnecessary icons or complex menus.
- **Snappiness:** Instant feedback on interactions.
- **Information Density:** See more notes with less scrolling.

## Development
- **Testing:** Run tests using `flutter test`.
- **Architecture:** Clean architecture with separation of concerns between UI, State, and Data layers.
