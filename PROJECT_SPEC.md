# PROJECT_SPEC.md - Flutter Notes App

## Overview
A clean, efficient, and professional-functional Notes application built with Flutter.

## Tech Stack
- **Framework:** Flutter
- **Language:** Dart
- **Storage:** SQLite (`sqflite`)
- **State Management:** Provider

## Features
- [x] **Create:** Add new notes with title and content.
- [x] **Read:** View a list of notes and open individual notes.
- [x] **Update:** Edit existing notes.
- [x] **Delete:** Remove notes with a confirmation prompt.

## Design
- Professional-Functional aesthetic.
- Inter font (via Google Fonts).
- Material 3 with custom color scheme.
